    нҺҳмқҙмҠӨ м»¬л§Ғ

    лЁёлҰҝмҶҚмңјлЎң 3D нҒҗлёҢ(м§ҒмңЎл©ҙмІҙ)лҘј л– мҳ¬лҰ¬кі  м–ҙлҠҗ л°©н–Ҙм—җм„ңл“  ліј мҲҳ мһҲлҠ” л©ҙмқҳ мөңлҢҖ к°ңмҲҳлҘј м„ём–ҙліҙм„ёмҡ”. мғҒмғҒл Ҙмқҙ н’Қл¶Җн•ҳм§Җ м•ҠлҚ”лқјлҸ„ м•„л§Ҳ мөңлҢҖ 3к°ңлқјлҠ” лӢөмқҙ лӮҳмҳ¬ кІҒлӢҲлӢӨ. нҒҗлёҢлҠ” м–ҙл–Ө мң„м№ҳлӮҳ л°©н–Ҙм—җм„ң ліҙлҚ”лқјлҸ„ 3к°ң мқҙмғҒмқҳ л©ҙмқ„ ліј мҲҳлҠ” м—ҶмҠөлӢҲлӢӨ. к·ёл ҮлӢӨл©ҙ ліҙмқҙм§ҖлҸ„ м•ҠлҠ” лӮҳлЁём§Җ 3к°ңмқҳ л©ҙмқ„ көімқҙ к·ёлҰҙ н•„мҡ”к°Җ мһҲмқ„к№Ңмҡ”? л§Ңм•Ҫ мқҙ л©ҙл“Өмқ„ м–ҙл–Ө л°©мӢқмңјлЎңл“  м ңкұ°н•  мҲҳ мһҲлӢӨл©ҙ, нҒҗлёҢмқҳ н”„лһҳк·ёлЁјнҠё м…°мқҙлҚ” мӢӨн–ү нҡҹмҲҳлҘј 50% мқҙмғҒ мӨ„мқј мҲҳ мһҲмқ„ кІҒлӢҲлӢӨ!

    50% лҢҖмӢ  50% мқҙмғҒмқҙлқјкі  н•ҳлҠ” мқҙмң лҠ” нҠ№м • к°ҒлҸ„м—җм„ңлҠ” л‘җ л©ҙ лҳҗлҠ” н•ң л©ҙл§Ң ліҙмқј мҲҳ мһҲкё° л•Ңл¬ёмһ…лӢҲлӢӨ. к·ёлҹ° кІҪмҡ°м—җлҠ” 50% мқҙмғҒмқ„ м Ҳм•Ҫн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    м •л§җ нӣҢлҘӯн•ң м•„мқҙл””м–ҙм§Җл§Ң, н•ҙкІ°н•ҙм•ј н•  л¬ём ңк°Җ н•ҳлӮҳ мһҲмҠөлӢҲлӢӨ. к°қмІҙмқҳ нҠ№м • л©ҙмқҙ ліҙлҠ” мӮ¬лһҢмқҳ мӢңм җм—җм„ң ліҙмқҙм§Җ м•ҠлҠ”м§Җ м–ҙл–»кІҢ м•Ң мҲҳ мһҲмқ„к№Ңмҡ”? лӢ«нһҢ лҸ„нҳ•мқ„ мғқк°Ғн•ҙ ліҙл©ҙ, к°Ғ л©ҙм—җлҠ” л‘җ к°ңмқҳ мёЎл©ҙ(л°©н–Ҙ)мқҙ мһҲмҠөлӢҲлӢӨ. к°Ғ мёЎл©ҙмқҖ мӮ¬мҡ©мһҗлҘј н–Ҙн•ҳкұ°лӮҳ мӮ¬мҡ©мһҗм—җкІҢ л“ұмқ„ ліҙмқҙкі  мһҲмЈ (л°”к№Ҙ л©ҙкіј м•ҲмӘҪ л©ҙ). л§Ңм•Ҫ ліҙлҠ” мӮ¬лһҢмқ„ н–Ҙн•ҳкі  мһҲлҠ” л©ҙл§Ң л ҢлҚ”л§Ғн•  мҲҳ мһҲлӢӨл©ҙ м–ҙл–Ёк№Ңмҡ”?

    мқҙкІғмқҙ л°”лЎң нҺҳмқҙмҠӨ м»¬л§Ғ(face culling)мқҳ мӣҗлҰ¬мһ…лӢҲлӢӨ. OpenGLмқҖ л·°м–ҙлҘј н–Ҙн•ҳлҠ” лӘЁл“  м •л©ҙ л©ҙмқ„ нҷ•мқён•ҳкі  л ҢлҚ”л§Ғн•ҳлҠ” л°ҳл©ҙ, нӣ„л©ҙмқ„ н–Ҙн•ҳлҠ” л©ҙмқҖ лӘЁл‘җ лІ„лҰјмңјлЎңмҚЁ н”„лһҳк·ёлЁјнҠё м…°мқҙлҚ” нҳём¶ң нҡҹмҲҳлҘј нҒ¬кІҢ мӨ„мһ…лӢҲлӢӨ. лӢЁ, м–ҙл–Ө л©ҙмқҙ м •л©ҙмқҙкі  м–ҙл–Ө л©ҙмқҙ нӣ„л©ҙмқём§Җ OpenGLм—җ м•Ңл ӨмӨҳм•ј н•©лӢҲлӢӨ. OpenGLмқҖ м •м җ лҚ°мқҙн„°мқҳ мҷҖмқёл”© мҲңм„ңлҘј л¶„м„қн•ҳлҠ” мҳҒлҰ¬н•ң л°©лІ•мқ„ мӮ¬мҡ©н•©лӢҲлӢӨ.

    мҷҖмқёл”© мҲңм„ң

    мӮјк°Ғнҳ•мқҳ м •м җмқ„ м •мқҳн•  л•Ң, мҡ°лҰ¬лҠ” мӢңкі„ л°©н–Ҙ лҳҗлҠ” л°ҳмӢңкі„ л°©н–ҘмңјлЎң нҠ№м • мҲңм„ңлЎң м •м җмқ„ м •мқҳн•©лӢҲлӢӨ. к°Ғ мӮјк°Ғнҳ•мқҖ 3к°ңмқҳ м •м җмңјлЎң кө¬м„ұлҗҳл©°, мӮјк°Ғнҳ•мқҳ мӨ‘мӢ¬м—җм„ң л°”лқјліё мҲңм„ңлҢҖлЎң мқҙ 3к°ңмқҳ м •м җмқ„ м§Җм •н•©лӢҲлӢӨ.

    мқҙлҜём§Җм—җм„ң ліҙмӢңлҠ” кІғмІҳлҹј лЁјм Җ мІ« лІҲм§ё м •м җмқ„ м •мқҳн•ҳкі , к·ё лӢӨмқҢ м •м җмқ„ 2лІҲ лҳҗлҠ” 3лІҲмңјлЎң м„ нғқн•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙ м„ нғқмқҙ мӮјк°Ғнҳ•мқҳ к°җкё° вҖӢвҖӢмҲңм„ңлҘј кІ°м •н•©лӢҲлӢӨ. лӢӨмқҢ мҪ”л“ңлҠ” мқҙлҘј ліҙм—¬мӨҚлӢҲлӢӨ.

    float vertices[] = {
    // мӢңкі„ л°©н–Ҙ
    vertices[0], // vertex 1
    vertices[1], // vertex 2
    vertices[2], // vertex 3
    // л°ҳмӢңкі„ л°©н–Ҙ
    vertices[0], // vertex 1
    vertices[2], // vertex 3
    vertices[1]  // vertex 2  
};

    мӮјк°Ғнҳ• кё°ліё мҡ”мҶҢлҘј кө¬м„ұн•ҳлҠ” к°Ғ 3к°ңмқҳ м •м җ м§‘н•©м—җлҠ” нҡҢм „ мҲңм„ңк°Җ нҸ¬н•Ёлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ. OpenGLмқҖ кё°ліё мҡ”мҶҢлҘј л ҢлҚ”л§Ғн•  л•Ң мқҙ м •ліҙлҘј мӮ¬мҡ©н•ҳм—¬ мӮјк°Ғнҳ•мқҙ м•һл©ҙ(front-facing) мӮјк°Ғнҳ•мқём§Җ л’·л©ҙ(back-facing) мӮјк°Ғнҳ•мқём§Җ нҢҗлӢЁн•©лӢҲлӢӨ. кё°ліём ҒмңјлЎң л°ҳмӢңкі„ л°©н–ҘмңјлЎң м •м җмқҙ м •мқҳлҗң мӮјк°Ғнҳ•мқҖ м•һл©ҙ мӮјк°Ғнҳ•мңјлЎң мІҳлҰ¬лҗ©лӢҲлӢӨ.

    м •м җ мҲңм„ңлҘј м •мқҳн•  л•ҢлҠ” н•ҙлӢ№ мӮјк°Ғнҳ•мқҙ л§Ҳм№ҳ лӮҳлҘј н–Ҙн•ҙ м„ң мһҲлҠ” кІғмІҳлҹј мӢңк°Ғнҷ”н•©лӢҲлӢӨ. л”°лқјм„ң м§Җм •н•ҳлҠ” к°Ғ мӮјк°Ғнҳ•мқҖ лӮҙк°Җ к·ё мӮјк°Ғнҳ•мқ„ м •л©ҙм—җм„ң л°”лқјліј л•Ң мӢңкі„ л°ҳлҢҖ л°©н–ҘмңјлЎң нҡҢм „н•ҙм•ј н•©лӢҲлӢӨ. мқҙл ҮкІҢ лӘЁл“  м •м җмқ„ м§Җм •н•ҳлҠ” л°©мӢқмқҳ мһҘм җмқҖ мӢӨм ң нҡҢм „ мҲңм„ңк°Җ лһҳмҠӨн„°нҷ” лӢЁкі„, мҰү м •м җ м…°мқҙлҚ”к°Җ мқҙлҜё мӢӨн–үлҗң нӣ„м—җ кі„мӮ°лҗңлӢӨлҠ” кІғмһ…лӢҲлӢӨ. к·ёлҹ¬л©ҙ м •м җл“ӨмқҖ ліҙлҠ” мӮ¬лһҢмқҳ мӢңм җм—җм„ң ліҙмқҙкІҢ лҗ©лӢҲлӢӨ.

    ліҙлҠ” мӮ¬лһҢмқҙ л°”лқјліҙлҠ” лӘЁл“  мӮјк°Ғнҳ• м •м җмқҖ мҡ°лҰ¬к°Җ м§Җм •н•ң мҲңм„ңлҢҖлЎң м •нҷ•н•ҳкІҢ л°°м—ҙлҗҳм–ҙ мһҲм§Җл§Ң, м •мңЎл©ҙмІҙ л°ҳлҢҖнҺём—җ мһҲлҠ” мӮјк°Ғнҳ•мқҳ м •м җмқҖ л°°м—ҙ мҲңм„ңк°Җ л°ҳм „лҗң нҳ•нғңлЎң н‘ңнҳ„лҗ©лӢҲлӢӨ. кІ°кіјм ҒмңјлЎң, мҡ°лҰ¬к°Җ л°”лқјліҙлҠ” мӮјк°Ғнҳ•мқҖ м•һл©ҙмқҙ ліҙмқҙлҠ” мӮјк°Ғнҳ•мңјлЎң, л’·л©ҙмқҳ мӮјк°Ғнҳ•мқҖ л’·л©ҙмқҙ ліҙмқҙлҠ” мӮјк°Ғнҳ•мңјлЎң лӮҳнғҖлӮ©лӢҲлӢӨ. лӢӨмқҢ мқҙлҜём§ҖлҠ” мқҙлҹ¬н•ң нҡЁкіјлҘј ліҙм—¬мӨҚлӢҲлӢӨ.

    м •м җ лҚ°мқҙн„°м—җм„ң л‘җ мӮјк°Ғнҳ•мқ„ лӘЁл‘җ мӢңкі„ л°ҳлҢҖ л°©н–Ҙ мҲңм„ң(м•һмӘҪ мӮјк°Ғнҳ•кіј л’ӨмӘҪ мӮјк°Ғнҳ•мқ„ к°Ғк°Ғ 1, 2, 3)лЎң м •мқҳн–ҲмҠөлӢҲлӢӨ. н•ҳм§Җл§Ң ліҙлҠ” мӮ¬лһҢмқҳ нҳ„мһ¬ мӢңм җм—җм„ң л’ӨмӘҪ мӮјк°Ғнҳ•мқ„ 1, 2, 3 мҲңм„ңлЎң к·ёлҰ¬л©ҙ л’ӨмӘҪ мӮјк°Ғнҳ•мқҙ мӢңкі„ л°©н–ҘмңјлЎң л ҢлҚ”л§Ғлҗ©лӢҲлӢӨ. л’ӨмӘҪ мӮјк°Ғнҳ•мқ„ мӢңкі„ л°ҳлҢҖ л°©н–Ҙ мҲңм„ңлЎң м§Җм •н–ҲмқҢм—җлҸ„ л¶Ҳкө¬н•ҳкі  мӢңкі„ л°©н–ҘмңјлЎң л ҢлҚ”л§ҒлҗҳлҠ” кІғмһ…лӢҲлӢӨ. мқҙлҠ” ліҙмқҙм§Җ м•ҠлҠ” л©ҙмқ„ м ңкұ°(м»¬л§Ғ)н•ҳл ӨлҠ” мҡ°лҰ¬мқҳ мқҳлҸ„мҷҖ м •нҷ•нһҲ мқјм№ҳн•ҳлҠ” кІ°кіјмһ…лӢҲлӢӨ!

    нҺҳмқҙмҠӨ м»¬л§Ғ

    мқҙ мһҘмқҳ мӢңмһ‘ л¶Җл¶„м—җм„ң OpenGLмқҖ л’·л©ҙмқ„ н–Ҙн•ҳлҠ” мӮјк°Ғнҳ•мңјлЎң л ҢлҚ”л§Ғлҗ  кІҪмҡ° н•ҙлӢ№ мӮјк°Ғнҳ• кё°ліё мҡ”мҶҢлҘј м ңкұ°н•  мҲҳ мһҲлӢӨкі  м–ёкёүн–ҲмҠөлӢҲлӢӨ. мқҙм ң м •м җмқҳ мҷҖмқёл”© мҲңм„ңлҘј м„Өм •н•ҳлҠ” л°©лІ•мқ„ м•Ңм•ҳмңјлҜҖлЎң кё°ліём ҒмңјлЎң л№„нҷңм„ұнҷ”лҗҳм–ҙ мһҲлҠ” OpenGLмқҳ нҺҳмқҙмҠӨ м»¬л§Ғ мҳөм…ҳмқ„ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    мқҙм „ мһҘм—җм„ң мӮ¬мҡ©н–ҲлҚҳ нҒҗлёҢ м •м җ лҚ°мқҙн„°лҠ” л°ҳмӢңкі„ л°©н–Ҙ к°җкё° мҲңм„ңлҘј кі л Өн•ҳм§Җ м•Ҡкі  м •мқҳлҗҳм—Ҳкё° л•Ңл¬ём—җ, л°ҳмӢңкі„ л°©н–Ҙ к°җкё° мҲңм„ңлҘј л°ҳмҳҒн•ҳлҸ„лЎқ м •м җ лҚ°мқҙн„°лҘј м—…лҚ°мқҙнҠён–ҲмҠөлӢҲлӢӨ. м—¬кё°м—җм„ң м—…лҚ°мқҙнҠёлҗң лҚ°мқҙн„°лҘј ліөмӮ¬н•ҳм—¬ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. к°Ғ мӮјк°Ғнҳ•мқҳ м •м җмқҙ лӘЁл‘җ л°ҳмӢңкі„ л°©н–ҘмңјлЎң м •мқҳлҗҳм–ҙ мһҲлӢӨлҠ” кІғмқ„ мӢңк°Ғнҷ”н•ҳлҠ” кІғмқҙ мўӢмҠөлӢҲлӢӨ.

    нҺҳмқҙмҠӨ м»¬л§Ғмқ„ нҷңм„ұнҷ”н•ҳл Өл©ҙ OpenGLмқҳ GL_CULL_FACE мҳөм…ҳмқ„ нҷңм„ұнҷ”н•ҳкё°л§Ң н•ҳл©ҙ лҗ©лӢҲлӢӨ.

    мқҙ мӢңм җл¶Җн„° м•һл©ҙмқҙ м•„лӢҢ лӘЁл“  л©ҙмқҖ лІ„л Өм§‘лӢҲлӢӨ(нҒҗлёҢ лӮҙл¶ҖлҘј лӮ м•„лӢӨлӢҲл©° лӘЁл“  лӮҙл¶Җ л©ҙмқҙ мӢӨм ңлЎң лІ„л Өм§ҖлҠ” кІғмқ„ нҷ•мқён•ҙ ліҙм„ёмҡ”). нҳ„мһ¬ OpenGLмқҙ л’·л©ҙмқ„ л ҢлҚ”л§Ғ м „м—җ лЁјм Җ лІ„лҰ¬лҸ„лЎқ нҢҗлі„н•ҳл©ҙ н”„лһҳк·ёлЁјнҠё л ҢлҚ”л§Ғ м„ұлҠҘмқҙ 50% мқҙмғҒ н–ҘмғҒлҗ©лӢҲлӢӨ(к·ёл Үм§Җ м•Ҡмңјл©ҙ к№Ҡмқҙ н…ҢмҠӨнҠём—җм„ң л ҢлҚ”л§Ғ нӣ„м—җ л’·л©ҙмқҙ лІ„л ӨмЎҢмқ„ кІғмһ…лӢҲлӢӨ). мқҙ л°©лІ•мқҖ нҒҗлёҢмҷҖ к°ҷмқҖ лӢ«нһҢ лҸ„нҳ•м—җм„ңл§Ң м ңлҢҖлЎң мһ‘лҸҷн•©лӢҲлӢӨ. мқҙм „ мһҘм—җм„ң лӢӨлЈ¬ н’ҖмһҺмқ„ к·ёлҰҙ л•ҢлҠ” м•һл©ҙкіј л’·л©ҙмқҙ лӘЁл‘җ ліҙм—¬м•ј н•ҳлҜҖлЎң л©ҙ м»¬л§Ғмқ„ лӢӨмӢң л№„нҷңм„ұнҷ”н•ҙм•ј н•©лӢҲлӢӨ.

    OpenGLмқ„ мӮ¬мҡ©н•ҳл©ҙ м»¬л§Ғн•  л©ҙмқҳ мң нҳ•мқ„ ліҖкІҪн•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. м•һл©ҙл§Ң м»¬л§Ғн•ҳкі  л’·л©ҙмқҖ м»¬л§Ғн•ҳм§Җ м•Ҡмңјл Өл©ҙ м–ҙл–»кІҢ н•ҙм•ј н• к№Ңмҡ”? glCullFaceлҘј мӮ¬мҡ©н•ҳм—¬ мқҙлҹ¬н•ң лҸҷмһ‘мқ„ м •мқҳн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    glCullFace(GL_FRONT);

    glCullFace н•ЁмҲҳм—җлҠ” м„ё к°Җм§Җ мҳөм…ҳмқҙ мһҲмҠөлӢҲлӢӨ.

    • GL_BACK: л’·л©ҙл§Ң м ңкұ°н•©лӢҲлӢӨ.
    • GL_FRONT: м „л©ҙ л©ҙл§Ң м ңкұ°н•©лӢҲлӢӨ.
    • GL_FRONT_AND_BACK: м•һл©ҙкіј л’·л©ҙ лӘЁл‘җлҘј м ңкұ°н•©лӢҲлӢӨ.

    glCullFaceмқҳ мҙҲкё°к°’мқҖ GL_BACKмһ…лӢҲлӢӨ. лҳҗн•ң glFrontFaceлҘј нҶөн•ҙ мӢңкі„ л°©н–Ҙ л©ҙмқ„ м•һл©ҙмңјлЎң мӮ¬мҡ©н•ҳлҠ” кІғмқ„ м„ нҳён•ҳлҸ„лЎқ OpenGLм—җ м•Ңл ӨмӨ„ мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.

    glFrontFace(GL_CCW);

    кё°ліёк°’мқҖ л°ҳмӢңкі„ л°©н–Ҙ м •л ¬мқ„ мқҳлҜён•ҳлҠ” GL_CCWмқҙл©°, лӢӨлҘё мҳөм…ҳмқҖ (лӢ№м—°нһҲ) мӢңкі„ л°©н–Ҙ м •л ¬мқ„ мқҳлҜён•ҳлҠ” GL_CWмһ…лӢҲлӢӨ.

    к°„лӢЁн•ң н…ҢмҠӨнҠёлЎң, OpenGLм—җкІҢ м•һл©ҙмқҳ мҲңм„ңк°Җ л°ҳмӢңкі„ л°©н–Ҙмқҙ м•„лӢҢ мӢңкі„ л°©н–ҘмңјлЎң кІ°м •лҗҳлҸ„лЎқ м•Ңл ӨмӨҢмңјлЎңмҚЁ к°җкё° мҲңм„ңлҘј л°ҳм „мӢңнӮ¬ мҲҳ мһҲмҠөлӢҲлӢӨ.

    glEnable(GL_CULL_FACE);
glCullFace(GL_BACK);
glFrontFace(GL_CW);

    к·ё кІ°кіј л’·л©ҙл§Ң л ҢлҚ”л§Ғлҗ©лӢҲлӢӨ.

    м°ёкі лЎң кё°ліё л°ҳмӢңкі„ л°©н–Ҙ к°җкё° мҲңм„ңлЎң м „л©ҙ л©ҙмқ„ м ңкұ°н•ҳл©ҙ лҸҷмқјн•ң нҡЁкіјлҘј м–»мқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

    glEnable(GL_CULL_FACE);
glCullFace(GL_FRONT);

    ліҙмӢңлӢӨмӢңн”ј, нҺҳмқҙмҠӨ м»¬л§ҒмқҖ мөңмҶҢн•ңмқҳ л…ёл ҘмңјлЎң OpenGL м• н”ҢлҰ¬мјҖмқҙм…ҳмқҳ м„ұлҠҘмқ„ н–ҘмғҒмӢңнӮӨлҠ” лҚ° л§Өмҡ° мң мҡ©н•ң лҸ„кө¬мһ…лӢҲлӢӨ. нҠ№нһҲ лӘЁл“  3D м• н”ҢлҰ¬мјҖмқҙм…ҳмқҙ кё°ліём ҒмңјлЎң мқјм •н•ң мҷҖмқёл”© мҲңм„ң(л°ҳмӢңкі„ л°©н–Ҙ)лЎң лӘЁлҚёмқ„ лӮҙліҙлӮҙкё° л•Ңл¬ём—җ лҚ”мҡұ нҡЁкіјм Ғмһ…лӢҲлӢӨ. лӢӨл§Ң, нҺҳмқҙмҠӨ м»¬л§Ғмқ„ нҶөн•ҙ м„ұлҠҘ н–ҘмғҒмқ„ м–»мқ„ к°қмІҙмҷҖ м»¬л§Ғ лҢҖмғҒм—җм„ң м ңмҷён•ҙм•ј н•  к°қмІҙлҘј кө¬л¶„н•ҳлҠ” кІғмқҙ мӨ‘мҡ”н•©лӢҲлӢӨ.

    м—°мҠөл¬ём ң

    • к°Ғ мӮјк°Ғнҳ•мқ„ мӢңкі„ л°©н–ҘмңјлЎң м§Җм •н•ҳм—¬ м •м җ лҚ°мқҙн„°лҘј мһ¬м •мқҳн•ң лӢӨмқҢ, мӢңкі„ л°©н–Ҙ мӮјк°Ғнҳ•мқ„ м „л©ҙмңјлЎң м„Өм •н•ҳм—¬ мһҘл©ҙмқ„ л ҢлҚ”л§Ғн•  мҲҳ мһҲмҠөлӢҲк№Ң? (н•ҙкІ° л°©лІ•)
