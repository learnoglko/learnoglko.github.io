    м…°мқҙлҚ”

    м•Ҳл…• мӮјк°Ғнҳ• мұ•н„°м—җм„ң м–ёкёүн–Ҳл“Ҝмқҙ, м…°мқҙлҚ”лҠ” GPUм—җм„ң мӢӨн–үлҗҳлҠ” мһ‘мқҖ н”„лЎңк·ёлһЁмһ…лӢҲлӢӨ. мқҙлҹ¬н•ң н”„лЎңк·ёлһЁмқҖ к·ёлһҳн”ҪмҠӨ нҢҢмқҙн”„лқјмқёмқҳ к°Ғ нҠ№м • лӢЁкі„м—җм„ң мӢӨн–үлҗ©лӢҲлӢӨ. к°„лӢЁнһҲ л§җн•ҙ, м…°мқҙлҚ”лҠ” мһ…л Ҙмқ„ м¶ңл ҘмңјлЎң ліҖнҷҳн•ҳлҠ” н”„лЎңк·ёлһЁм—җ м§ҖлӮҳм§Җ м•ҠмҠөлӢҲлӢӨ. лҳҗн•ң м…°мқҙлҚ”лҠ” м„ңлЎң нҶөмӢ н•  мҲҳ м—ҶлҸ„лЎқ л§Өмҡ° лҸ…лҰҪм ҒмңјлЎң мһ‘лҸҷн•ҳлҠ” н”„лЎңк·ёлһЁмһ…лӢҲлӢӨ. м…°мқҙлҚ” к°„мқҳ нҶөмӢ мқҖ мҳӨм§Ғ мһ…л Ҙкіј м¶ңл Ҙмқ„ нҶөн•ҙм„ңл§Ң мқҙлЈЁм–ҙм§‘лӢҲлӢӨ.

    мқҙм „ мһҘм—җм„ңлҠ” м…°мқҙлҚ”мқҳ кё°ліём Ғмқё к°ңл…җкіј мҳ¬л°”лҘё мӮ¬мҡ©лІ•мқ„ к°„лһөн•ҳкІҢ мӮҙнҺҙліҙм•ҳмҠөлӢҲлӢӨ. мқҙм ң м…°мқҙлҚ”, нҠ№нһҲ OpenGL м…°мқҙл”© м–ём–ҙм—җ лҢҖн•ҙ ліҙлӢӨ мқјл°ҳм Ғмқё л°©мӢқмңјлЎң м„ӨлӘ…н•ҳкІ мҠөлӢҲлӢӨ.

    GLSL

    м…°мқҙлҚ”лҠ” CмҷҖ мң мӮ¬н•ң м–ём–ҙмқё GLSLлЎң мһ‘м„ұлҗ©лӢҲлӢӨ. GLSLмқҖ к·ёлһҳн”Ҫ мһ‘м—…м—җ мөңм Ғнҷ”лҗҳм–ҙ мһҲмңјл©°, лІЎн„° л°Ҹ н–үл ¬ мЎ°мһ‘м—җ нҠ№нҷ”лҗң мң мҡ©н•ң кё°лҠҘл“Өмқ„ нҸ¬н•Ён•ҳкі  мһҲмҠөлӢҲлӢӨ.

    м…°мқҙлҚ”лҠ” н•ӯмғҒ лІ„м „ м„ м–ёмңјлЎң мӢңмһ‘н•ҳкі , к·ё л’Өм—җ мһ…л Ҙ л°Ҹ м¶ңл Ҙ ліҖмҲҳ лӘ©лЎқ, мң лӢҲнҸј(uniforms) ліҖмҲҳ, к·ёлҰ¬кі  л©”мқё н•ЁмҲҳк°Җ мқҙм–ҙм§‘лӢҲлӢӨ. к°Ғ м…°мқҙлҚ”мқҳ м§„мһ…м җмқҖ л©”мқё н•ЁмҲҳмқҙл©°, м—¬кё°м„ң мһ…л Ҙ ліҖмҲҳлҘј мІҳлҰ¬н•ҳкі  кІ°кіјлҘј м¶ңл Ҙ ліҖмҲҳм—җ м¶ңл Ҙн•©лӢҲлӢӨ. мң лӢҲнҸј ліҖмҲҳк°Җ л¬ҙм—Үмқём§Җ лӘЁлҘҙлҚ”лқјлҸ„ кұұм •н•ҳм§Җ л§Ҳм„ёмҡ”. кі§ м„ӨлӘ…л“ңлҰ¬кІ мҠөлӢҲлӢӨ.

    м…°мқҙлҚ”лҠ” мқјл°ҳм ҒмңјлЎң лӢӨмқҢкіј к°ҷмқҖ кө¬мЎ°лҘј к°Җм§Җкі  мһҲмҠөлӢҲлӢӨ.

    #version version_number
in type in_variable_name;
in type in_variable_name;

out type out_variable_name;

uniform type uniform_name;

void main()
{
  // мһ…л Ҙк°’мқ„ мІҳлҰ¬н•ҳкі  м—¬лҹ¬к°Җм§Җ к·ёлһҳн”Ҫ мһ‘м—… мҲҳн–ү
  ...
  // мІҳлҰ¬лҗң кІ°кіјлҘј м¶ңл Ҙ ліҖмҲҳм—җ м¶ңл Ҙ
  out_variable_name = weird_stuff_we_processed;
}

    м •м җ м…°мқҙлҚ”м—җ лҢҖн•ҙ кө¬мІҙм ҒмңјлЎң мқҙм•јкё°н•  л•Ң, к°Ғ мһ…л Ҙ ліҖмҲҳлҠ” м •м җ мҶҚм„ұ(vertex attribute)мқҙлқјкі лҸ„ н•©лӢҲлӢӨ. м„ м–ён•  мҲҳ мһҲлҠ” м •м җ мҶҚм„ұмқҳ мөңлҢҖ к°ңмҲҳлҠ” н•ҳл“ңмӣЁм–ҙм—җ мқҳн•ҙ м ңн•ңлҗ©лӢҲлӢӨ. OpenGLмқҖ н•ӯмғҒ мөңмҶҢ 16к°ңмқҳ 4м„ұл¶„ м •м җ мҶҚм„ұмқ„ мӮ¬мҡ©н•  мҲҳ мһҲлҸ„лЎқ ліҙмһҘн•ҳм§Җл§Ң, мқјл¶Җ н•ҳл“ңмӣЁм–ҙм—җм„ңлҠ” лҚ” л§ҺмқҖ мҶҚм„ұмқ„ н—Ҳмҡ©н•  мҲҳ мһҲмңјл©°, мқҙлҠ” GL_MAX_VERTEX_ATTRIBSлҘј мҝјлҰ¬н•ҳм—¬ нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    int nrAttributes;
glGetIntegerv(GL_MAX_VERTEX_ATTRIBS, &nrAttributes);
std::cout << "Maximum nr of vertex attributes supported: " << nrAttributes << std::endl;

    мқҙ н•ЁмҲҳлҠ” лҢҖк°ң мөңмҶҢк°’мқё 16мқ„ л°ҳнҷҳн•ҳлҠ”лҚ°, мқҙлҠ” лҢҖл¶Җл¶„мқҳ лӘ©м Ғм—җ м¶©л¶„н•  кІғмһ…лӢҲлӢӨ.

    нғҖмһ…

    GLSLмқҖ лӢӨлҘё н”„лЎңк·ёлһҳл°Қ м–ём–ҙмҷҖ л§Ҳм°¬к°Җм§ҖлЎң ліҖмҲҳмқҳ мў…лҘҳлҘј м§Җм •н•ҳлҠ” лҚ°мқҙн„° нғҖмһ…мқ„ к°Җм§Җкі  мһҲмҠөлӢҲлӢӨ. GLSLм—җлҠ” C м–ём–ҙм—җм„ң нқ”нһҲ ліј мҲҳ мһҲлҠ” int, float, double, uint, boolкіј к°ҷмқҖ кё°ліё нғҖмһ…мқҙ лҢҖл¶Җл¶„ нҸ¬н•Ёлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ. лҳҗн•ң, лІЎн„°мҷҖ н–үл ¬мқҙлқјлҠ” л‘җ к°Җм§Җ м»Ён…Ңмқҙл„Ҳ нғҖмһ…мқ„ л§Һмқҙ мӮ¬мҡ©н•ҳкІҢ лҗ  кІғмһ…лӢҲлӢӨ. н–үл ¬м—җ лҢҖн•ҙм„ңлҠ” мқҙнӣ„ мһҘм—җм„ң мһҗм„ёнһҲ лӢӨлЈЁкІ мҠөлӢҲлӢӨ.

    лІЎн„°

    GLSLм—җм„ң лІЎн„°лҠ” л°©кёҲ м–ёкёүн•ң кё°ліё мң нҳ•л“Өмқ„ лӢҙмқ„ мҲҳ мһҲлҠ” 2, 3 лҳҗлҠ” 4к°ңмқҳ кө¬м„ұ мҡ”мҶҢлЎң мқҙлЈЁм–ҙм§„ м»Ён…Ңмқҙл„Ҳмһ…лӢҲлӢӨ. лІЎн„°лҠ” лӢӨмқҢкіј к°ҷмқҖ нҳ•нғңлҘј м·Ён•  мҲҳ мһҲмҠөлӢҲлӢӨ(nмқҖ кө¬м„ұ мҡ”мҶҢмқҳ к°ңмҲҳлҘј лӮҳнғҖлғ…лӢҲлӢӨ).

    • vecn: nк°ңмқҳ л¶ҖлҸҷмҶҢмҲҳм җ(float) к°’лЎң мқҙлЈЁм–ҙм§„ кё°ліё лІЎн„°.
    • bvecn: nк°ңмқҳ л¶ҲлҰ¬м–ё к°’мңјлЎң мқҙлЈЁм–ҙм§„ лІЎн„°.
    • ivecn: nк°ңмқҳ м •мҲҳ к°’мңјлЎң мқҙлЈЁм–ҙм§„ лІЎн„°.
    • uvecn: nк°ңмқҳ л¶Җнҳё м—ҶлҠ” м •мҲҳ(unsigned integer) к°’мңјлЎң мқҙлЈЁм–ҙм§„ лІЎн„°.
    • dvecn: nк°ңмқҳ лҚ”лё”(double) к°’мңјлЎң мқҙлЈЁм–ҙм§„ лІЎн„°.

    лҢҖл¶Җл¶„мқҳ кІҪмҡ° л¶ҖлҸҷмҶҢмҲҳм җ мҲ«мһҗк°Җ мҡ°лҰ¬мқҳ лӘ©м Ғм—җ м¶©л¶„н•ҳкё° л•Ңл¬ём—җ кё°ліём Ғмқё vecnмқ„ мӮ¬мҡ©н•ҳкІҢ лҗ  кІғмһ…лӢҲлӢӨ.

    лІЎн„°мқҳ кө¬м„ұ мҡ”мҶҢлҠ” vec.xмҷҖ к°ҷмқҙ м ‘к·јн•  мҲҳ мһҲмңјл©°, м—¬кё°м„ң xлҠ” лІЎн„°мқҳ мІ« лІҲм§ё кө¬м„ұ мҡ”мҶҢмһ…лӢҲлӢӨ. .x, .y, .z, .wлҘј мӮ¬мҡ©н•ҳл©ҙ к°Ғк°Ғ мІ« лІҲм§ё, л‘җ лІҲм§ё, м„ё лІҲм§ё, л„Ө лІҲм§ё кө¬м„ұ мҡ”мҶҢм—җ м ‘к·јн•  мҲҳ мһҲмҠөлӢҲлӢӨ. GLSLм—җм„ңлҠ” мғүмғҒм—җлҠ” rgbaлҘј, н…ҚмҠӨмІҳ мўҢн‘ңм—җлҠ” stpqлҘј мӮ¬мҡ©н•ҳм—¬ лҸҷмқјн•ң кө¬м„ұ мҡ”мҶҢм—җ м ‘к·јн•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.

    лІЎн„° лҚ°мқҙн„° нғҖмһ…мқҖ мҠӨмң„мҰҗл§Ғ(swizzling)мқҙлқјлҠ” нқҘлҜёлЎӯкі  мң м—°н•ң кө¬м„ұ мҡ”мҶҢ м„ нғқ кё°лҠҘмқ„ м ңкіөн•©лӢҲлӢӨ. мҠӨмң„мҰҗл§Ғмқ„ мӮ¬мҡ©н•ҳл©ҙ лӢӨмқҢкіј к°ҷмқҖ кө¬л¬ёмқ„ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    vec2 someVec;
vec4 differentVec = someVec.xyxx;
vec3 anotherVec = differentVec.zyw;
vec4 otherVec = someVec.xxxx + anotherVec.yxzy;

    мӣҗліё лІЎн„°м—җ н•ҙлӢ№ кө¬м„ұ мҡ”мҶҢк°Җ мһҲлҠ” н•ң, мөңлҢҖ 4к°ңмқҳ л¬ёмһҗлҘј мЎ°н•©н•ҳм—¬ мғҲ лІЎн„°(лҸҷмқјн•ң нғҖмһ…)лҘј мғқм„ұн•  мҲҳ мһҲмҠөлӢҲлӢӨ. н•ҳм§Җл§Ң мҳҲмӢңлЎң vec2мқҳ .z кө¬м„ұ мҡ”мҶҢм—җлҠ” м ‘к·јн•  мҲҳ м—ҶмҠөлӢҲлӢӨ. лҳҗн•ң лІЎн„°лҘј лӢӨлҘё лІЎн„° мғқм„ұмһҗ нҳём¶ңмқҳ мқёмҲҳлЎң м „лӢ¬н•ҳм—¬ н•„мҡ”н•ң мқёмҲҳмқҳ мҲҳлҘј мӨ„мқј мҲҳ мһҲмҠөлӢҲлӢӨ.

    vec2 vect = vec2(0.5, 0.7);
vec4 result = vec4(vect, 0.0, 0.0);
vec4 otherResult = vec4(result.xyz, 1.0);

    лІЎн„°лҠ” лӘЁл“  мў…лҘҳмқҳ мһ…л Ҙкіј м¶ңл Ҙм—җ мӮ¬мҡ©н•  мҲҳ мһҲлҠ” мң м—°н•ң лҚ°мқҙн„° мң нҳ•мһ…лӢҲлӢӨ. мқҙ мұ… м „мІҙм—җм„ң лІЎн„°лҘј м°Ҫмқҳм ҒмңјлЎң нҷңмҡ©н•ҳлҠ” лӢӨм–‘н•ң мҳҲм ңлҘј ліҙкІҢ лҗ  кІғмһ…лӢҲлӢӨ.

    мқём•Өм•„мӣғ

    н–„лІ„кұ°к°Җ м•„лӢҲлқј мқён’Ӣ(input)кіј м•„мӣғн’Ӣ(output)мһ…лӢҲлӢӨ!

    мң лӢҲнҸј

    мң лӢҲнҸј(Uniforms)мқҖ CPUм—җм„ң мӢӨн–үлҗҳлҠ” м• н”ҢлҰ¬мјҖмқҙм…ҳмқҳ лҚ°мқҙн„°лҘј GPUмқҳ м…°мқҙлҚ”лЎң м „лӢ¬н•ҳлҠ” лҳҗ лӢӨлҘё л°©лІ•мһ…лӢҲлӢӨ. н•ҳм§Җл§Ң мң лӢҲнҸјмқҖ м •м җ мҶҚм„ұкіјлҠ” м•Ҫк°„мқҳ м°Ёмқҙк°Җ мһҲмҠөлӢҲлӢӨ. мІ«м§ё, мң лӢҲнҸјмқҖ м „м—ӯ ліҖмҲҳмһ…лӢҲлӢӨ. м „м—ӯ ліҖмҲҳлһҖ м…°мқҙлҚ” н”„лЎңк·ёлһЁ к°қмІҙлі„лЎң кі мң н•ң к°’мқ„ к°Җм§Җл©°, н”„лЎңк·ёлһЁмқҳ м–ҙлҠҗ лӢЁкі„м—җм„ңл“  лӘЁл“  м…°мқҙлҚ”м—җм„ң м ‘к·јн•  мҲҳ мһҲмқҢмқ„ мқҳлҜён•©лӢҲлӢӨ. л‘ҳм§ё, мң лӢҲнҸј к°’мқ„ м–ҙл–Ө к°’мңјлЎң м„Өм •н•ҳл“ , н•ҙлӢ№ к°’мқҖ мһ¬м„Өм •лҗҳкұ°лӮҳ м—…лҚ°мқҙнҠёлҗ  л•Ңк№Ңм§Җ мң м§Җлҗ©лӢҲлӢӨ.

    GLSLм—җм„ң мң лӢҲнҸј ліҖмҲҳлҘј м„ м–ён•ҳл Өл©ҙ м…°мқҙлҚ”м—җ uniform нӮӨмӣҢл“ңмҷҖ нғҖмһ…, мқҙлҰ„мқ„ м¶”к°Җн•ҳкё°л§Ң н•ҳл©ҙ лҗ©лӢҲлӢӨ. мқҙл ҮкІҢ н•ҳл©ҙ м…°мқҙлҚ”м—җм„ң мғҲлЎң м„ м–ёлҗң мң лӢҲнҸј ліҖмҲҳлҘј мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлІҲм—җлҠ” мң лӢҲнҸј ліҖмҲҳлҘј нҶөн•ҙ мӮјк°Ғнҳ•мқҳ мғүмғҒмқ„ м„Өм •н•  мҲҳ мһҲлҠ”м§Җ мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ.

    #version 330 core
out vec4 FragColor;

{
    FragColor = ourColor;
}

    нҳ„мһ¬ мң лӢҲнҸјмқҖ л№„м–ҙ мһҲмҠөлӢҲлӢӨ. м•„м§Ғ мң лӢҲнҸјм—җ м•„л¬ҙ лҚ°мқҙн„°лҸ„ м¶”к°Җн•ҳм§Җ м•Ҡм•ҳмңјлӢҲ, мқҙм ң м¶”к°Җн•ҙ ліҙкІ мҠөлӢҲлӢӨ. лЁјм Җ м…°мқҙлҚ”м—җм„ң мң лӢҲнҸј мҶҚм„ұмқҳ мқёлҚұмҠӨ/мң„м№ҳлҘј м°ҫм•„м•ј н•©лӢҲлӢӨ. мң лӢҲнҸјмқҳ мқёлҚұмҠӨ/мң„м№ҳлҘј м°ҫм•ҳмңјл©ҙ к°’мқ„ м—…лҚ°мқҙнҠён•  мҲҳ мһҲмҠөлӢҲлӢӨ. н”„лһҳк·ёлЁјнҠё м…°мқҙлҚ”м—җ лӢЁмқј мғүмғҒмқ„ м „лӢ¬н•ҳлҠ” лҢҖмӢ , мӢңк°„мқҙ м§ҖлӮЁм—җ л”°лқј мғүмғҒмқҙ м җм§„м ҒмңјлЎң ліҖн•ҳлҸ„лЎқ л§Ңл“Өм–ҙ ліҙкІ мҠөлӢҲлӢӨ.

glUniform4f(vertexColorLocation, 0.0f, greenValue, 0.0f, 1.0f);

    лЁјм Җ glfwGetTime() н•ЁмҲҳлҘј мӮ¬мҡ©н•ҳм—¬ мӢӨн–ү мӢңк°„мқ„ мҙҲ лӢЁмң„лЎң к°Җм ёмҳөлӢҲлӢӨ. к·ёлҹ° лӢӨмқҢ sin н•ЁмҲҳлҘј мӮ¬мҡ©н•ҳм—¬ мғүмғҒмқ„ 0.0~1.0 лІ”мң„лЎң ліҖнҷ”мӢңнӮӨкі  к·ё кІ°кіјлҘј greenValueм—җ м ҖмһҘн•©лӢҲлӢӨ.

    лӢӨмқҢмңјлЎң glGetUniformLocation н•ЁмҲҳлҘј мӮ¬мҡ©н•ҳм—¬ ourColor мң лӢҲнҸјмқҳ мң„м№ҳлҘј вҖӢвҖӢмЎ°нҡҢн•©лӢҲлӢӨ. мЎ°нҡҢ н•ЁмҲҳм—җлҠ” м…°мқҙлҚ” н”„лЎңк·ёлһЁкіј мң„м№ҳлҘј к°Җм ёмҳ¬ мң лӢҲнҸјмқҳ мқҙлҰ„мқ„ м „лӢ¬н•©лӢҲлӢӨ. glGetUniformLocation н•ЁмҲҳк°Җ -1мқ„ л°ҳнҷҳн•ҳл©ҙ мң„м№ҳлҘј м°ҫмқ„ мҲҳ м—ҶлӢӨлҠ” мқҳлҜёмһ…лӢҲлӢӨ. л§Ҳм§Җл§үмңјлЎң glUniform4f н•ЁмҲҳлҘј мӮ¬мҡ©н•ҳм—¬ мң лӢҲнҸј к°’мқ„ м„Өм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мң лӢҲнҸј мң„м№ҳлҘј м°ҫлҠ” лҚ°м—җлҠ” м…°мқҙлҚ” н”„лЎңк·ёлһЁмқ„ лЁјм Җ мӮ¬мҡ©н•  н•„мҡ”к°Җ м—Ҷм§Җл§Ң, мң лӢҲнҸј к°’мқ„ м—…лҚ°мқҙнҠён•ҳл Өл©ҙ лЁјм Җ н•ҙлӢ№ н”„лЎңк·ёлһЁмқ„ мӮ¬мҡ©н•ҙм•ј н•©лӢҲлӢӨ(glUseProgram нҳём¶ң). мқҙлҠ” нҳ„мһ¬ нҷңм„ұнҷ”лҗң м…°мқҙлҚ” н”„лЎңк·ёлһЁм—җ мң лӢҲнҸј к°’мқ„ м„Өм •н•ҳкё° л•Ңл¬ёмһ…лӢҲлӢӨ.

    OpenGLмқҖ ліём§Ҳм ҒмңјлЎң C лқјмқҙлёҢлҹ¬лҰ¬мқҙкё° л•Ңл¬ём—җ н•ЁмҲҳ мҳӨлІ„лЎңл”©мқ„ кё°ліём ҒмңјлЎң м§Җмӣҗн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ. л”°лқјм„ң н•ЁмҲҳлҘј м—¬лҹ¬ мң нҳ•мңјлЎң нҳём¶ңн•  мҲҳ мһҲлҠ” кІҪмҡ°, OpenGLмқҖ н•„мҡ”н•ң к°Ғ мң нҳ•м—җ лҢҖн•ҙ мғҲлЎңмҡҙ н•ЁмҲҳлҘј м •мқҳн•©лӢҲлӢӨ. glUniform н•ЁмҲҳк°Җ л°”лЎң к·ё лҢҖн‘ңм Ғмқё мҳҲмһ…лӢҲлӢӨ. мқҙ н•ЁмҲҳлҠ” м„Өм •н•ҳл ӨлҠ” мң лӢҲнҸј мң нҳ•м—җ л§һлҠ” нҠ№м • м ‘лҜёмӮ¬лҘј н•„мҡ”лЎң н•©лӢҲлӢӨ. к°ҖлҠҘн•ң м ‘лҜёмӮ¬ лӘҮ к°Җм§ҖлҠ” лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

    • f: мқҙ н•ЁмҲҳлҠ” к°’мңјлЎң л¶ҖлҸҷ мҶҢмҲҳм җ(float)мқ„ кё°лҢҖн•©лӢҲлӢӨ.
    • i: мқҙ н•ЁмҲҳлҠ” к°’мңјлЎң м •мҲҳ(int)лҘј кё°лҢҖн•©лӢҲлӢӨ.
    • ui: мқҙ н•ЁмҲҳлҠ” к°’мңјлЎң л¶Җнҳё м—ҶлҠ” м •мҲҳ(unsigned int)лҘј кё°лҢҖн•©лӢҲлӢӨ.
    • 3f: мқҙ н•ЁмҲҳлҠ” к°’мңјлЎң 3к°ңмқҳ л¶ҖлҸҷ мҶҢмҲҳм җ мҲ«мһҗлҘј кё°лҢҖн•©лӢҲлӢӨ.
    • fv: мқҙ н•ЁмҲҳлҠ” к°’мңјлЎң floatнҳ• лІЎн„° лҳҗлҠ” л°°м—ҙмқ„ кё°лҢҖн•©лӢҲлӢӨ.

    OpenGL мҳөм…ҳмқ„ м„Өм •н•ҳл Өл©ҙ н•ҙлӢ№ мң нҳ•м—җ л§һлҠ” мҳӨлІ„лЎңл“ңлҗң н•ЁмҲҳлҘј м„ нғқн•ҳкё°л§Ң н•ҳл©ҙ лҗ©лӢҲлӢӨ. мқҙ кІҪмҡ° мң лӢҲнҸј ліҖмҲҳм—җ 4к°ңмқҳ float к°’мқ„ к°ңлі„м ҒмңјлЎң м„Өм •н•ҳл Өкі  н•ҳлҜҖлЎң glUniform4f н•ЁмҲҳлҘј нҶөн•ҙ лҚ°мқҙн„°лҘј м „лӢ¬н•©лӢҲлӢӨ(fv лІ„м „мқ„ мӮ¬мҡ©н•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ).

    мқҙм ң мң лӢҲнҸј ліҖмҲҳмқҳ к°’мқ„ м„Өм •н•ҳлҠ” л°©лІ•мқ„ м•Ңм•ҳмңјлӢҲ, мқҙлҘј л ҢлҚ”л§Ғм—җ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. мғүмғҒмқҙ м җм§„м ҒмңјлЎң ліҖн•ҳлҸ„лЎқ н•ҳл Өл©ҙ л§Ө н”„л Ҳмһ„л§ҲлӢӨ мқҙ мң лӢҲнҸј ліҖмҲҳлҘј м—…лҚ°мқҙнҠён•ҙм•ј н•©лӢҲлӢӨ. к·ёл Үм§Җ м•Ҡмңјл©ҙ н•ң лІҲл§Ң м„Өм •н•ҳл©ҙ мӮјк°Ғнҳ•мқҙ лӢЁмғүмңјлЎң мң м§Җлҗ©лӢҲлӢӨ. л”°лқјм„ң greenValueлҘј кі„мӮ°н•ҳкі  л§Ө л ҢлҚ”л§Ғ л°ҳліөл§ҲлӢӨ мң лӢҲнҸј ліҖмҲҳлҘј м—…лҚ°мқҙнҠён•©лӢҲлӢӨ.

{
    // мһ…л Ҙ мІҳлҰ¬
    processInput(window);

    // л ҢлҚ”л§Ғ
    // мғүмғҒ лІ„нҚјлҘј мҙҲкё°нҷ”

    glClear(GL_COLOR_BUFFER_BIT);

    // м…°мқҙлҚ” н”„лЎңк·ёлһЁмқ„ нҷңм„ұнҷ”н•ҳлҠ” кІғмқ„ мһҠм§Җ л§Ҳм„ёмҡ”.

    мқҙ мҪ”л“ңлҠ” мқҙм „ мҪ”л“ңлҘј л№„көҗм Ғ к°„лӢЁн•ҳкІҢ ліҖнҳ•н•ң кІғмһ…лӢҲлӢӨ. мқҙлІҲм—җлҠ” мӮјк°Ғнҳ•мқ„ к·ёлҰ¬кё° м „м—җ л§Ө н”„л Ҳмһ„л§ҲлӢӨ uniform ліҖмҲҳлҘј м—…лҚ°мқҙнҠён•©лӢҲлӢӨ. uniform ліҖмҲҳлҘј мҳ¬л°”лҘҙкІҢ м—…лҚ°мқҙнҠён•ҳл©ҙ мӮјк°Ғнҳ•мқҳ мғүмғҒмқҙ л…№мғүм—җм„ң кІҖмқҖмғүмңјлЎң, лӢӨмӢң л…№мғүмңјлЎң м җм°Ё ліҖн•ҳлҠ” кІғмқ„ ліј мҲҳ мһҲмҠөлӢҲлӢӨ.     л§үнһҲлҠ” л¶Җл¶„мқҙ мһҲмңјл©ҙ м—¬кё°м—җм„ң мҶҢмҠӨ мҪ”л“ңлҘј нҷ•мқён•ҳм„ёмҡ”.

    ліҙмӢңлӢӨмӢңн”ј, мң лӢҲнҸјмқҖ л§Ө н”„л Ҳмһ„л§ҲлӢӨ ліҖкІҪлҗ  мҲҳ мһҲлҠ” мҶҚм„ұмқ„ м„Өм •н•ҳкұ°лӮҳ м• н”ҢлҰ¬мјҖмқҙм…ҳкіј м…°мқҙлҚ” к°„м—җ лҚ°мқҙн„°лҘј көҗнҷҳн•ҳлҠ” лҚ° мң мҡ©н•ң лҸ„кө¬мһ…лӢҲлӢӨ. н•ҳм§Җл§Ң к°Ғ м •м җм—җ мғүмғҒмқ„ м„Өм •н•ҳл Өл©ҙ м–ҙл–»кІҢ н•ҙм•ј н• к№Ңмҡ”? мқҙ кІҪмҡ° м •м җмқҳ к°ңмҲҳл§ҢнҒј мң лӢҲнҸјмқ„ м„ м–ён•ҙм•ј н•©лӢҲлӢӨ. лҚ” лӮҳмқҖ н•ҙкІ°мұ…мқҖ м •м җ мҶҚм„ұм—җ лҚ” л§ҺмқҖ лҚ°мқҙн„°лҘј нҸ¬н•Ён•ҳлҠ” кІғмқҙл©°, м§ҖкёҲл¶Җн„° к·ё л°©лІ•мқ„ мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ.

    лҚ” л§ҺмқҖ мҶҚм„ұ!

    мқҙм „ мһҘм—җм„ң VBOлҘј мұ„мҡ°кі , м •м җ мҶҚм„ұ нҸ¬мқён„°лҘј кө¬м„ұн•ҳкі , мқҙ лӘЁл“  кІғмқ„ VAOм—җ м ҖмһҘн•ҳлҠ” л°©лІ•мқ„ мӮҙнҺҙліҙм•ҳмҠөлӢҲлӢӨ. мқҙлІҲм—җлҠ” м •м җ лҚ°мқҙн„°м—җ мғүмғҒ лҚ°мқҙн„°лҸ„ м¶”к°Җн•ҳл Өкі  н•©лӢҲлӢӨ. м •м җ л°°м—ҙм—җ 3к°ңмқҳ floatнҳ• мғүмғҒ лҚ°мқҙн„°лҘј м¶”к°Җн•  кІғмһ…лӢҲлӢӨ. мӮјк°Ғнҳ•мқҳ к°Ғ м •м җм—җ к°Ғк°Ғ л№Ёк°•, мҙҲлЎқ, нҢҢлһ‘ вҖӢвҖӢмғүмғҒмқ„ н• лӢ№н•©лӢҲлӢӨ.
