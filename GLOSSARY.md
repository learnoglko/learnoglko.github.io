## 좌표계 / 수학

- Normal : 법선


## 렌더링 파이프라인

- Framebuffer : 프레임버퍼
- Renderbuffer : 렌더버퍼
- Attachment : 어태치먼트
- Attach : 첨부, 상황에 따라 연결 등 다른 단어
- Fragment : 프래그먼트
- Vertex : 정점
- Vertex Shader : 정점 셰이더
- Normalized Device Coordinates : 원문 직역(정규화된 장치 좌표 또는 NDC)
- Graphics Pipeline : 그래픽스 파이프라인
- Unifrom : 유니폼
- unifrom-scaling : 균일 스케일링

## 조명
- Ambient lighting : 주변 조명
- Diffuse lighting : 확산 조명
- Specular lighting : 반사 조명
- Ambient (변수명) : ambient (번역하지 않음)

## 변수명

1. 코드 내부의 실제 변수명은 번역하지 않는다.
2. 설명 문장에서 변수의 의미를 설명할 때만 자연스럽게 번역한다.
3. 변수명을 임의로 한국어로 바꾸어 재정의하지 않는다.

## 기타

영어 단어를 알아야 할 것 같은 경우(명확한 한글 번역이 널리 퍼지지 않았거나, 발음을 음차했거나 하는 경우 등)에는 영단어를 괄호 안에 적는다.

법전을 번역하는거나 출판할 책이 아니므로 편하게 번역하도록.
